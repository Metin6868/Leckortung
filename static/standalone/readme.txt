SCHADENSBERICHT APP - OFFLINE VERSION
====================================

Diese Version der Schadensbericht-App ist eine reine HTML/JavaScript-Anwendung, die vollständig im Browser läuft, ohne einen Server zu benötigen.

ANLEITUNG:
---------

1. STARTEN DER APP:
   - Öffnen Sie die Datei "index.html" in einem modernen Browser (Chrome, Firefox, Safari, Edge)
   - Die App sollte sofort funktionieren, ohne Installation

2. FUNKTIONEN:
   - Erstellen, Bearbeiten und Löschen von Schadensberichten
   - Bilder hinzufügen (direkt vom Gerät)
   - PDF-Export mit anpassbarem Layout
   - Alle Daten werden lokal auf Ihrem Gerät gespeichert (IndexedDB)

3. SPEICHERUNG:
   - Alle Daten werden im Browser gespeichert
   - Sie bleiben auch nach dem Schließen des Browsers erhalten
   - WICHTIG: Wenn Sie den Browsercache löschen, werden auch Ihre Daten gelöscht!

4. KOMPATIBILITÄT:
   - Diese App funktioniert auf Desktop-Computern, Laptops, Tablets und Smartphones
   - Die App benötigt einen modernen Browser (nicht Internet Explorer)
   - Bei Problemen bitte die neueste Version Ihres Browsers installieren

5. HINWEISE:
   - Auf mobilen Geräten können Sie Bilder direkt mit der Kamera aufnehmen
   - Die Benutzeroberfläche passt sich automatisch an verschiedene Bildschirmgrößen an
   - Für optimale PDF-Qualität empfehlen wir Chrome oder Edge

Bei Fragen wenden Sie sich bitte an Ihren Administrator.

----
Diese App wurde mit moderner Webtechnologie entwickelt und benötigt keine Internetverbindung.
